# Question 1: RESTful Asset Management API

## How to Run

1. Open terminal in this folder
2. Run: `bal run`
3. The server will start on http://localhost:9090

## API Endpoints

We will implement:
- GET /assets - Get all assets
- POST /assets - Create new asset
- GET /assets/{assetTag} - Get specific asset
- PUT /assets/{assetTag} - Update asset
- DELETE /assets/{assetTag} - Delete asset
